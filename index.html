About Lyrics Writer

How it works

Lyrics Writer helps you write song lyrics with real-time phonetic feedback and rhyme detection.

  • Type a lyric on each line. Press Enter to move to the next line.
  • IPA phonetics appear below each line, showing how the words are pronounced using the International Phonetic Alphabet.
  • Rhyme detection automatically finds lines that end with the same sounds. Matching endings are bolded and color-coded — lines sharing a rhyme get a colored left border.
  • Syllable count is shown as a small badge on each line.
  • Drag & drop lines using the ≡ handle to reorder them.
  • Paste multiple lines at once — they are automatically split into separate lyric lines.
  • Undo / Redo with Ctrl+Z / Ctrl+Shift+Z or the header buttons.
  • Your work is auto-saved in your browser and survives page refreshes.

🇬🇧 English

IPA is fetched from a free dictionary API. Common words (pronouns, verbs, prepositions, contractions) are available instantly via a built-in dictionary. Words not found in either source are converted using English pronunciation rules — results are approximate given English's irregular spelling.

🇫🇷 French

French uses a three-layer approach: a large built-in dictionary of common words, the free dictionary API, and a comprehensive rule-based converter as fallback. French spelling is fairly regular, so the rule-based converter handles most unknown words accurately. Missing accents are tolerated — typing ecrire will produce the same result as écrire.

🇪🇸 Spanish

Spanish IPA comes from the dictionary API and a built-in dictionary of common words. Spanish has very regular pronunciation, so the API coverage combined with the built-in dictionary covers most vocabulary.

🇮🇹 Italian

Italian works like Spanish: dictionary API plus a built-in dictionary. Italian pronunciation is largely predictable from spelling, making the API results reliable for most words.

🏛️ Latin

Latin uses a fully rule-based converter following Classical Latin pronunciation. Since Latin orthography maps deterministically to pronunciation, every word is converted without needing an API. Digraphs (ae, oe, au, ph, th, ch, qu) and all standard letter values are handled.

Rhymes are detected by comparing the IPA endings of all lines. The longer the shared ending, the stronger the rhyme. Cross-language rhymes work naturally since comparison is based on sound, not spelling.
